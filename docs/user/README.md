## Choosing GUI vs CLI

Both the GUI app and command line work equally well. Choose based on your preference:

| Preference | Recommendation |
|------------|----------------|
| New to version control | Start with **GUI app** |
| Comfortable with Terminal | **CLI** is faster |
| Working remotely (SSH) | **CLI** only option |
| Prefer keyboard shortcuts | **CLI** |
| Prefer point-and-click | **GUI app** |
